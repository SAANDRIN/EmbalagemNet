# Aplicativo EmbalagemNet

_**- Aplicativo criado com o intuito de simular um app de vendas de algumas embalagens, onde as tecnologias exploradas são: os componentes de interface, recyclerView, NestedScrowView, ScrowView e SQLite (prática e conceitos).**_
_**- As formas de pagamento são apenas uma simulação, onde é possível ao final da "venda", gerar um pdf que pode ser enviado direto pelo whatsappp ou exportado para o dispositivo**_
